# Clustering-Assignment


This repository contains my solutions for the **Clustering Assignment**, completed as part of my Machine Learning learning journey. The notebook includes theoretical explanations, conceptual questions, and practical implementations related to clustering algorithms and unsupervised learning techniques.

---

##  Assignment Overview

The objective of this assignment is to understand the fundamentals of **Clustering**, an unsupervised machine learning technique used to group similar data points based on their characteristics. The assignment covers clustering concepts, popular algorithms, evaluation methods, advantages, limitations, and real-world applications.
